About Glue API Migration Guides

The Glue API migration guides provide step-by-step implementation on how to manually update a module to a newer version.

If you have spotted an issue with either of the guides, please feel free to create an issue or pull request by using the Edit on GitHub option.

Not found a migration guide you are looking for? Let us know about it by creating an issue on Github or dropping an email to academy@spryker.com.
