Adyen - Filter Payment Methods

Adyen module provides filtering available payment methods depend on result of /paymentMethods API call.

To enable this add \SprykerEco\Zed\Adyen\Communication\Plugin\AdyenPaymentMethodFilterPlugin to filter plugins list at src/Pyz/Zed/Payment/PaymentDependencyProvider.php:


protected function getPaymentMethodFilterPlugins()
{
	return [
		...
		new AdyenPaymentMethodFilterPlugin(),
	];
