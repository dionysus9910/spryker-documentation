Wishlist Items Management

addItem(WishlistItemTransfer $wishlistItemTransfer)

Adds item to wishlist.

Required values: fkCustomer, fkProduct. Optional: wishlistName. In case wishlist name is not provided the default value will be used.

<?php
$wishlistItemTransfer = (new WishlistItemTransfer())
    ->setWishlistName('foobar')
    ->setFkCustomer(1)
    ->setFkProduct(2);

$wishlistItemTransfer = $wishlistFacade->addItem($wishlistItemTransfer);

removeItem(WishlistItemTransfer $wishlistItemTransfer)

Removes item from wishlist.

Required values: fkCustomer, fkProduct. Optional: wishlistName. In case wishlist name is not provided the default value will be used.

<?php
$wishlistItemTransfer = (new WishlistItemTransfer())
    ->setWishlistName('foobar')
    ->setFkCustomer(1)
    ->setFkProduct(2);

$wishlistItemTransfer = $wishlistFacade->removeItem ($wishlistItemTransfer);

addItemCollection(WishlistTransfer $wishlistTransfer, array $wishlistItemCollection)

Adds collection of items to a wishlist.

Required values of WishlistTransfer: fkCustomer, name.

Required values of WishlistItemTransfer: fkProduct.

<?php
$wishlistTransfer = (new WishlistTransfer())
    ->setFkCustomer(1)
    ->setName('foobar');

$item1 = (new WishlistItemTransfer())
    ->setFkProduct(1);
$item1 = (new WishlistItemTransfer())
    ->setFkProduct(2);

$itemCollection = [$item1, $item2];
$wishlistFacade->addItemCollection($wishlistTransfer, $itemCollection);

emptyWishlist(WishlistTransfer $wishlistTransfer)

Removes all wishlist items.

Required values: idWishlist.

<?php
$wishlistTransfer = (new WishlistTransfer())
    ->setIdWishlist(1);

$wishlistFacade->emptyWishlist($wishlistTransfer);
