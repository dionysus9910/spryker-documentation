Kernel

The Kernel provides all needed abstract classes to be able to build new Modules which can be used in the Spryker context.

Located class instance cache.

By default, a call to e.g. $locator->moduleName()->facade() will always return a new instance of the module's facade. You can easily turn on the located class instance cache by overriding the Spryker\Shared\Kernel\KernelConfig::isLocatorInstanceCacheEnabled() in your project and return true.
