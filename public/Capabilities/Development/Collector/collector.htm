Collector

Used to be: http://spryker.github.io/core/bundles/collector/

The Collector module provides functionality to aggregate data from an SQL database, and synchronize with a NoSQL data store, in a format that can be easily consumed by front-end application.

See Performance and Scalability for an overview of the separation of responsibilities between Yves and Zed.
