Company Account Feature Integration

The Company Account feature is shipped with following modules:

Module Description
Company Provides basic infrastructure for handling companies
CompanyGui Responsible for managing companies in Zed (enabling/disabling, status manage)
CompanyBusinessUnit Bears the logic for creation and management of the company organization (business units)
CompanyBusinessUnitGui Responsible for managing company business units in the Admin Zed UI
CompanyRole Provides infrastructure for handling company user roles and permissions (managing user roles and role permissions)
CompanyUnitAddress Provides basic infrastructure for handling company and business unit addresses
CompanyUnitAddressGui Responsible for managing company addresses in the Admin Zed UI
CompanyUser Implements functionality for Company User creation
CompanyPage Provides Yves pages for Shop
CompanyMailConnector Provides basic infrastructure for handling emails (registration, company status change etc.)
CompanyUnitAddressLabel Provides the support of address labels to addresses of company business units.
CompanyUnitAddressGui Module is responsible for visual representation of company unit address labels in Zed.

To install the feature, follow the steps below.

1. Install necessary modules using composer

Update existing and install the required modules:

composer update "spryker/*" "spryker-shop/*"

composer require spryker/company-extension:"^1.0.0" spryker/company:"^1.0.0" spryker/company-gui:"^1.0.0" spryker/company-user-extension:"^1.0.0" spryker/company-user:"^2.0.0" spryker/company-business-unit-extension:"^1.0.0" spryker/company-business-unit:"^2.0.0" spryker/company-unit-address-extension:"^1.0.0" spryker/company-unit-address:"^1.0.0" spryker/permission-extension:"^1.0.0" spryker/permission spryker/company-role:"^1.1.1" spryker/company-mail-connector:"^1.0.0" spryker-shop/company-page:"^1.1.0" spryker/company-business-unit-gui-extension:"^1.0.0" spryker/company-business-unit-gui:"^1.0.0" spryker/company-unit-address-gui-extension:"^1.0.0" spryker/company-unit-address-gui:"^1.0.0" spryker/company-unit-address-label:"^1.0.0" spryker/company-unit-address-label-gui:"^1.0.0" spryker-shop/company-user-invitation-widget:"^1.0.0" --update-with-dependencies

2. Add plugins to Zed's CompanyDependencyProvider
Module Plugin Description Method in Dependency Provider
CompanyBusinessUnit CompanyBusinessUnitCreatePlugin Creates default Business unit after company registration getCompanyPostCreatePlugins
CompanyRole CompanyRoleCreatePlugin Creates default Admin company role and assigns all permissions to it. After an initial user has been created, this role will be assigned to him/her. getCompanyPostCreatePlugins
CompanyUser CompanyUserCreatePlugin Creates initial company user getCompanyPostCreatePlugins
CompanyMail SendCompanyStatusChangePlugin Sends company status change email to initial company user getCompanyPostSavePlugins
src/Pyz/Zed/Company/CompanyDependencyProvider.php 

namespace Pyz\Zed\Company;
 
use Spryker\Zed\Company\CompanyDependencyProvider as SprykerCompanyDependencyProvider;
use Spryker\Zed\CompanyBusinessUnit\Communication\Plugin\Company\CompanyBusinessUnitCreatePlugin;
use Spryker\Zed\CompanyMailConnector\Communication\Plugin\Company\SendCompanyStatusChangePlugin;
use Spryker\Zed\CompanyRole\Communication\Plugin\CompanyRoleCreatePlugin;
use Spryker\Zed\CompanyUser\Communication\Plugin\Company\CompanyUserCreatePlugin;
 
class CompanyDependencyProvider extends SprykerCompanyDependencyProvider
{
    /**
     * &return \Spryker\Zed\CompanyExtension\Dependency\Plugin\CompanyPostSavePluginInterface[]
     */
    protected function getCompanyPostSavePlugins(): array
    {
        return [
            new SendCompanyStatusChangePlugin()
        ];
    }
 
    /**
     * &return \Spryker\Zed\CompanyExtension\Dependency\Plugin\CompanyPostCreatePluginInterface[]
     */
    protected function getCompanyPostCreatePlugins(): array
    {
        return [
            new CompanyBusinessUnitCreatePlugin(),
            new CompanyRoleCreatePlugin(),
            new CompanyUserCreatePlugin()
        ];
    }
}
