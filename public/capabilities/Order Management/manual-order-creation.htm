Manual Order Creation

MC Multi-currency

Manual Order Entry, allows shop administrators to add orders directly from the Administration Interface. It comes in handy when customers want to make an order but can not do it for some reason, or when preparing offers for customers, or when shop employees are taking orders while on the phone or during an e-mail conversation with the customer - in these cases shop employees can manually place an order by filling out the order creation form in the dedicated area of the Administration Interface.

If you are:

Developer

Shop Administrator

   
   

Allow your sales agents to create orders manually. In the Administration Interface users can effortlessly create orders, enter the relevant checkout information and place the order on behalf the customer. The order entry form, out of the box, is optimized for processing orders via phone and can be customized as required by adding new fields from other modules. A sales agent can easily modify the prices in the manual order creation workflow, to accommodate customer specific prices or discounts.

Shop Administrators able to create an order, fulfill with cart details and checkout information and place it.

  • Order entry from is optimized for sales manager that creating order by contacting customer by phone.
  • Process represent one page checkout.
  • Ability to force prices for order items (base item price)
  • Order entry form is extendible with new fields from other modules
