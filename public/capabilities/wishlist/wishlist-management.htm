Wishlist

createWishlist(WishlistTransfer $wishlistTransfer)

Creates wishlist for a specific customer with given name.

Required values: name, fkCustomer.

<?php
$wishlistTransfer = (new WishlistTransfer())
    ->setName('foobar')
    ->setFkCustomer(1);

$wishlistTransfer = $wishlistFacade->createWishlist($wishlistTransfer);

updateWishlist(WishlistTransfer $wishlistTransfer)

Updates specific wishlist.

Required values: idWishlist.

<?php
$wishlistTransfer = (new WishlistTransfer())
    ->setName('new name')
    ->setIdWishlist(1);

$wishlistTransfer = $wishlistFacade->updateWishlist($wishlistTransfer);

removeWishlist(WishlistTransfer $wishlistTransfer)

Removes wishlist and its items.

Required values: idWishlist.

<?php
$wishlistTransfer = (new WishlistTransfer())
    ->setIdWishlist(1);

$wishlistTransfer = $wishlistFacade->removeWishlist($wishlistTransfer);

getWishlistByName(WishlistTransfer $wishlistTransfer)

Returns wishlist by specific name for a given customer.

Required values: fkCustomer, name.

<?php
$wishlistTransfer = (new WishlistTransfer())
    ->setFkCustomer(1)
    ->setName('foobar');

$wishlistTransfer = $wishlistFacade->getWishlistByName($wishlistTransfer);

getWishlistOverview(WishlistOverviewRequestTransfer $wishlistOverviewRequestTransfer)

Returns wishlist by specific name for a given customer, with paginated items.

Pagination is controlled with page, itemsPerPage, orderBy and orderDirection values of WishlistOverviewRequestTransfer.

Required values of WishlistTransfer: fkCustomer, name.

Required values of WishlistOverviewRequestTransfer: WishlistTransfer.

<?php

$wishlistTransfer = (new WishlistTransfer())
    ->setName('foobar')
    ->setFkCustomer(1);

$wishlistOverviewRequest = (new WishlistOverviewRequestTransfer())
    ->setWishlist($wishlistTransfer)
    ->setPage(1)
    ->setItemsPerPage(10)
    ->setOrderBy('created_at')
    ->setOrderDirection('DESC');

$wishlistOverviewRequestTransfer = $wishlistFacade->getWishlistOverview($wishlistOverviewRequest);
