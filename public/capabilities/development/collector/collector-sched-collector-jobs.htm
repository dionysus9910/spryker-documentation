Scheduling Collector Jobs

The cronjob file jobs.php stores the configuration related to collectors. 

<?php
$jobs[] = [
    'name' => 'export-search',
    'command' => '$PHP_BIN vendor/bin/console collector:search:export',
    'schedule' => '*/10 * * * *',
    'enable' => false,
    'stores' => $allStores,
];
?>

In the example above, a cronjob is configured to run every 10 minutes and update the search data store.

Check the jobs.php file or Cronjob Scheduling for details.
