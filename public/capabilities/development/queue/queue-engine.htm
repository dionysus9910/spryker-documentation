Default Queue Engine

The Spryker virtual machine is shipped with a ready to use RabbitMQ engine inside.

To access the management UI for Demoshop:

  1. Go to http://zed.de.demoshop.local:15672/ (Replace demoshop with your project name if you customized it)
  2. Enter the default credentials: user: admin Password: mate20mg
  3. Click Login

For information on how to work with RabbitMQ, see https://www.rabbitmq.com/tutorials/tutorial-one-php.html or From the CMD, run: 1 man rabbitmq-server
