Queue Job Configuration

You can also find the Worker configuration in jobs.php as part of the Demopshop and adjust the worker command to run every minute.

To adjust the worker command to run every minute: 

<?php

$jobs[] = [
    'name' => 'queue-worker-start',
    'command' => '$PHP_BIN vendor/bin/console queue:worker:start -vvv',
    'schedule' => '* * * * *',
    'enable' => true,
    'run_on_non_production' => true,
    'stores' => $allStores,
];

?>
