Discount
The Discount module is responsible for managing discounts in the Spryker OS.

Overview

In the diagram below you can see how the discount details are stored in the database:

<a href=”https://www.gliffy.com/go/share/sayj0ui56yd2ce1e25yn </a>

The spy_discount_voucher_pool table purpose is to group the vouchers codes generated for each discount. You can deactivate a group of voucher codes by deactivating the voucher pool to which they are linked.

If you have a multi-store environment, you can set up discounts for specific countries. See the migration guide to obtain instructions on how to enable the Discount Per Store feature.
