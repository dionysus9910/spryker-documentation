  • Deletes only file info version with a given id

  • Returns a file meta info and a file content with a specified version

findMimeType

saveMimeType

  • Creates or updates mime type
deleteMimeType

  • Deletes mime type
findAllowedMimeTypes

  • Finds allowed mime types
