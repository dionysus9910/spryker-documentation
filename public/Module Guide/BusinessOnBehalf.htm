BusinessOnBehalf

 

Latest version: 1.0.3 View on GitHub
Last update: 08 Nov 2018
 
  Type: Core Splitted
Group: <none>

API (Facades)

Facade Method Description
expandCustomerWithIsOnBehalf

  • Sets IsOnBehalf property as true when the provided customer has multiple company users connected.

  • Sets IsOnBehalf property as false otherwise.

  • Uses provided customer ID to find company users.

  • Ignores Company user/Customer activity flags.
findActiveCompanyUsersByCustomerId

  • Retrieves a collection of active company users related to the provided customer.

  • Uses customer ID to find company users.

  • Hydrates company transfer to company user transfer.
setDefaultCompanyUser

  • Deselects default company user is_default option.

  • Makes sent company user to be a default one.
unsetDefaultCompanyUserByCustomer

  • Removes company user default flags for the provided customer.
setDefaultCompanyUserToCustomer

  • Finds a company user by idCustomer and is_default = true

  • Doesn't set anything when a default record does not exist

  • Doesn't set anything when CustomerTransfer::companyUserTransfer is not null

Clients

Client Method Description
findActiveCompanyUsersByCustomerId

  • Retrieves a collection of active company users related to the provided customer.

  • Uses customer ID to find company users.

  • Hydrates company transfer to company user transfer.
setDefaultCompanyUser

  • Sets is_default to true for provided company user

  • Removes is_default flag for all other company/business unit pairs of a customer
unsetDefaultCompanyUser

  • Removes all isDefault flags from company users that belong to the provided customer

  • Returns customer transfer object with company user set as null

Dependencies:

  • PHP >=7.1
  • Company ^1.0.0
  • CompanyUser ^1.2.0 || ^2.0.0
  • Kernel ^3.0.0
  • ZedRequest ^3.0.0

Suggested Installations:

  • Customer If you want to hydrate customer with isOnBehalf property and default company user: ^7.10.0.
