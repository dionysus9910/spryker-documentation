Money

Money module makes it easier to work with monetary values. The key feature of this module is to convert a MoneyTransfer into the accurate string version of it, given the current locale and currency.

 

Latest version: 2.4.3 View on GitHub
Last update: 24 Oct 2018
 
  Type: Core Splitted
Group: Util

API (Facades)

Facade Method Description
fromInteger

  • Converts int amount and isoCode to MoneyTransfer Object

  • If isoCode is not provided it will use from Store configured one
fromFloat

  • Converts float amount and isoCode to MoneyTransfer Object

  • If isoCode is not provided it will use from Store configured one
fromString

  • Converts string amount and isoCode to MoneyTransfer Object

  • If isoCode is not provided it will use from Store configured one
formatWithSymbol

  • Converts MoneyTransfer Object into string representation with currency symbol
formatWithoutSymbol

  • Converts MoneyTransfer Object into string representation without currency symbol
parse

  • Parses a formatted string representation to MoneyTransfer
convertIntegerToDecimal

  • Converts an integer value into decimal value
convertDecimalToInteger

  • Converts a decimal value into integer value

Dependencies:

  • Moneyphp/Money ^3.0.0
  • PHP >=7.1
  • Currency ^2.0.0 || ^3.1.0
  • Kernel ^3.11.0
  • Store ^1.1.0
  • Symfony ^3.0.0

Suggested Installations:

  • Silex If you want to use ServiceProvider.
  • Twig If you want to use Money in Twig.
