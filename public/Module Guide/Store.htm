Store

Store contains store related functionality. It provides store related information on request. Store can deliver current store information, information for all stores and information for a store requested by id. For a store currencies and locales can be retrieved.

 

Latest version: 1.9.0 View on GitHub
Last update: 05 Oct 2018
 
  Type: Core Splitted
Group: Functional

API (Facades)

Facade Method Description
getCurrentStore

  • Returns currently selected store transfer
getAllStores

  • Reads all active stores and returns list of transfers
getStoreById

  • Read store by primary id from database
getStoreByName

  • Read store by store name
getStoresWithSharedPersistence

  • Reads all shared store from Store transfer and populates data from configuration.

  • The list of stores with which this store shares database, the value is store name.
getCountries

  • Returns store countries

Clients

Client Method Description
getCurrentStore

  • Retrieves the current Store as a transfer object.
getStoreByName

  • Retrieves a Store as a transfer object.

Dependencies:

  • PHP >=7.1
  • Kernel ^3.18.0
  • PropelOrm ^1.0.0
  • Symfony ^3.0.0

Suggested Installations:

  • Quote If you want to use StoreQuoteTransferExpander plugin, minimum required version: 1.2.0
