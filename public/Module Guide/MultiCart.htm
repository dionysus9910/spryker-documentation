API (Facades)

Facade Method Description
setDefaultQuote

  • Mark quote as default.

  • Mark all other customer quotes as not default.
initDefaultCustomerQuote

  • Finds customer quotes.

  • Mark first quote as default, if all customers quote are not default.
expandQuoteResponse

  • Adds customer quote collection to quote response transfer after cart operation handling.

  • Replace quote with active quote if it exist.
resetQuoteDefaultFlagByCustomer

  • Mark all customer quotes as not default.
resolveQuoteName

  • Resolve quote name to make it unique for customer.
getQuoteCollectionByCriteria

  • Returns the quotes collection by provided criteria.

  • Collection is filtered by customer reference.

  • Pagination and ordering options can be passed to criteria.

Clients

Client Method Description
getDefaultCart

  • Gets active quote.
setDefaultQuote

  • Mark quote as active.

  • Mark all other customer carts as inactive.
getQuoteCollection

  • Gets customer quote collection.
setQuoteCollection

  • Set customer quote collection.
findQuoteById

  • Find quote by id in customer session.
isMultiCartAllowed

  • Checks if multicart functionality allowed.
createQuote

  • Create new quote.

  • Save quote in database.
updateQuote

  • Update quote name.
deleteQuote

  • Remove quote.
duplicateQuote

  • Create copy of quote with.

  • Mark new quote as active.

  • Add duplicate suffix new quote name.

  • Save quote in database.
clearQuote

  • Remove all items from quote.

  • Save quote in database.
getQuoteCollectionByCriteria

  • Returns quote collection by criteria from Zed.

  • Collection is filtered by customer reference.

  • Pagination and ordering options can be passed to criteria.
isQuoteDeletable

  • Сhecks the possibility of removing the quote

  • If customer has more one quote - returns TRUE
