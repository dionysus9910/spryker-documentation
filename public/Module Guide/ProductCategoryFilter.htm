ProductCategoryFilter

 

Latest version: 1.2.2 View on GitHub
Last update: 05 Oct 2018
 
  Type: Core Splitted
Group: <none>

API (Facades)

Facade Method Description
createProductCategoryFilter

  • Persist new product category filter entity into database.

  • The returned transfer contains the ID of the created product category filter.

  • Touches "product_category_filter" entity as active.
updateProductCategoryFilter

  • Updates existing product category filter in database.

  • Touches "product_category_filter" entity as active.
findProductCategoryFilterByCategoryId

  • Finds existing product category filter in database.

  • Returns the product category filter.
getAllProductCategoriesWithFilters

  • Returns an array of product category IDs that have specific filters attached to them
deleteProductCategoryFilterByCategoryId

  • Removes existing product category filter from database.

  • Touches "product_category_Filter" as deleted.

Clients

Client Method Description
updateFacetsByCategory

  • Returns formatted facets based on product category filters from array
updateCategoryFacets

  • Returns formatted facets based on product category filters from category id and locale name
getProductCategoryFiltersForCategoryByLocale

  • Returns saved product category filters from storage

Dependencies:

  • PHP >=7.1

  • PropelOrm ^1.0.0
