API (Facades)

Facade Method Description
postSaveHook

  • Executes a post save hook for the following payment methods:

    Sofort / authorize: checks for an external redirect URL in transaction log and redirects customer to the payment system
processExternalPaymentResponse

  • Prepares the request for ExternalResponseTransaction

  • Executes ExternalResponseTransaction and returns the results
authorizePayment

  • Sends payment authorize request to Heidelpay gateway.

  • Saves the transaction result in DB for future recognition
debitPayment

  • Sends payment debit request to Heidelpay gateway.

  • Saves the transaction result in DB for future recognition
capturePayment

  • Sends payment capture request to Heidelpay gateway.

  • Saves the transaction result in DB for future recognition
saveOrderPayment

  • Saves order payment method data according to quote and checkout response transfer data.
getPaymentByIdSalesOrder

  • Fetches order payment by fk_sales_order
getAuthorizeTransactionLog

  • Fetches authorise transaction for a given order by idSalesOrder
getCreditCardPaymentOptions

  • Calculates available payment options for Credit Card payment method based on a given quote object

Clients

Client Method Description
translateErrorMessageByCode

  • Retrieves current locale;

  • Calls external library (heidelpay/php-customer-message) to get error message for customer, based on

    provided error code and locale.
getAuthorizeTransactionLogForOrder

  • Fetches from Zed transaction log for a given order by it's reference
getCreditCardPaymentOptions

  • For a given quote, requests from Zed list of allowed payment options for credit card payment method
processExternalPaymentResponse

  • Sends external response from payment provider (POST request) to Zed for processing
saveCreditCardRegistration

  • Sends credit card registration request to Zed for saving
findRegistrationByIdAndQuote

  • tries to find credit card registration by registration id and customer quote to reassure,

    that credit card registration really belongs to current customer
parseExternalResponse

  • Parses the external Heidelpay array response, transforming it to the transfer object
getQuoteFromSession

  • Retrieve quote from current customer session
filterResponseParameters

  • Filter response from heidelpay from unnecessary params.
