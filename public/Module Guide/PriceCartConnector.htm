PriceCartConnector

PriceCartConnector is a cart expander plugin where the price is provided for an item which is added to the cart.

 

Latest version: 4.6.0 View on GitHub
Last update: 01 Nov 2018
 
  Type: Core Splitted
Group: Supportive

API (Facades)

Facade Method Description
addPriceToItems

  • Adds product prices to item, based on currency, price mode and price type.
validatePrices

  • Validates product prices, checks if prices are valid for current currency, price mode, price type combination

  • Writes error message to response transfer if not valid.
filterItemsWithoutPrice

  • Removes items without price from quote.

  • Adds note to messages about removed items.

Dependencies:

  • PHP >=7.1
  • CartExtension ^2.0.0
  • Kernel ^3.0.0
  • Messenger ^3.0.0
  • Price ^5.0.0
  • PriceProduct ^1.3.0 || ^2.3.0

Suggested Installations:

  • Cart If you want to use Cart plugins.
