API (Facades)

Facade Method Description
findPermissionsByIdCompanyUser

  • Get permissions for customer company user.
expandQuoteResponseWithSharedCarts

  • Adds customer shared cart to QuoteResponseTransfer.
installSharedCartPermissions

  • Add base shared quote permission group list to database.
getQuotePermissionGroupList

  • Get filtered quote permission groups list.
updateQuoteShareDetails

  • Update quote share details for quote.
resetQuoteDefaultFlagByCustomer

  • Reset is_default flag for all quotes shared with customer.
quoteSetDefault

  • Mark share connection for quote and customer as default.
deleteShareForQuote

  • Remove all share connection for quote.
expandCustomer

  • Add quote permissions for customer company user to customer.
isSharedQuoteDefault

  • Checks if shared cart default for company user.
getShareDetailsByIdQuote

  • Returns share details collection of quote by quote id.
deleteShareRelationsForCompanyUserId

  • Un-shares quotes for company user.

Clients

Client Method Description
getQuotePermissionGroupList

  • Get quote permission group list.
addShareCart

  • Adds share information to quote.

  • Updates quote in database.
removeShareCart

  • Remove share information from quote.

  • Updates quote in database.
getQuoteAccessLevel

  • Calculate current customer permission on given quote.
updateQuotePermissions

  • Updates quote with permissions from share details.

  • Sends Zed request to update quote.
getShareDetailsByIdQuoteAction

  • Returns share detail collection by quote id.

  • Retrieves ShareDetails collection from Zed.
isQuoteDeletable

  • Сhecks the possibility of removing the quote.

  • If customer isn't owner of quote but has permission for write this quote - return TRUE

  • If customer has another quote where he is owner - return TRUE
dismissSharedCart

  • Sends Zed Request to get share detail collection by quote id.

  • Filters quote share detail from share details by company user id.

  • Sends Zed request to update quote.

Dependencies:
