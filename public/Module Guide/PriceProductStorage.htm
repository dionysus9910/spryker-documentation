 2.3.0

API (Facades)

Facade Method Description
publishPriceProductConcrete

  • Queries all priceProduct with the given productConcreteIds

  • Stores data as json encoded to storage table

  • Sends a copy of data to queue based on module config
unpublishPriceProductConcrete

  • Finds and deletes priceProduct storage entities with the given productConcreteIds

  • Sends delete message to queue based on module config
publishPriceProductAbstract

  • Queries all priceProduct with the given productAbstractIds

  • Stores data as json encoded to storage table

  • Sends a copy of data to queue based on module config
unpublishPriceProductAbstract

  • Finds and deletes priceProduct storage entities with the given productAbstractIds

  • Sends delete message to queue based on module config

Clients

Client Method Description
getPriceProductAbstractTransfers

  • Returns abstract product prices from Storage.
getPriceProductConcreteTransfers

  • Returns concrete product prices from Storage.
getResolvedPriceProductConcreteTransfers

  • Returns a resolved concrete product price from storage.

  • Returns product concrete price data if it exists.

  • Returns product abstract price data otherwise.
