Oauth

 

Latest version: 1.1.0 View on GitHub
Last update: 03 Nov 2018
 
  Type: Core Splitted
Group: <none>

API (Facades)

Facade Method Description
processAccessTokenRequest

  • Process token request

  • Returns new access token when user provider return valid user

  • Executes scope plugins

  • Executes user plugins

  • Saves issues access token in database for auditing
validateAccessToken

  • Validates access JWT token against signature and token
saveScope

  • Creates new scope in database
saveClient

  • Creates new client in database
findScopeByIdentifier

  • Retrieves a oauth scope using the identifier within the provided transfer.
findClientByIdentifier

  • Retrieves a oauth client using the identifier within the provided transfer.
getScopesByIdentifiers

  • Retrieves a oauth scopes using the identifiers.

Clients

Client Method Description
processAccessTokenRequest

  • Process access tokens request, makes RPC Zed call to retrieve new access token
validateAccessToken

  • Validates JWT token, checks if fingerprint is valid using public key, does not go to Zed.

Dependencies:

  • League/Oauth2-Server ^7.1
  • PHP >=7.1
  • Config ^3.0.0
  • Guzzle ^2.0.0
  • Kernel ^3.0.0
  • OauthExtension ^1.0.0
  • ZedRequest ^3.0.0
