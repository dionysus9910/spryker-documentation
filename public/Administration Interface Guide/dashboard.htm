Dashboard

Dashboard is the first page administrator sees when logging in to the Administration Interface. The dashboard provides a quick overview of basic order-related statistics. Specifically, this page presents you with a summary of the following information:

Order counts

Order statuses

Top products

Count Orders The very first chart you see on the Dashboard page is Count Orders. This chart provides a summary of orders made for the last 7 days. X axis shows days, and Y axis shows number of orders.
Status Orders The very first chart you see on the Dashboard page is Count Orders. This chart provides a summary of orders made for the last 7 days. X axis shows days, and Y axis shows number of orders.
Top Products The Top Products chart visualizes 10 most sold products. X axis shows placed orders, and Y axis shows product names.
