Merchants

To open this page, go to Merchants -> Merchants

This page is where you can add and manage merchants. Merchants are actually sellers usually represented by a company.

From this page you can:

  • Create merchants
  • Edit merchants
  • Delete merchants

Graphic User Interface

The Merchants page consists of the following elements:

# Element Description
1 +Add Merchant This button adds a new merchant. See Managing Merchants for more details.
2 Show [number] entries Here you can select how many items you want to see in the list of navigation elements (10, 25, 50, 100).
3 Search Search and filter merchants. Start typing merchant ID or name to filter the results and display them by Merchant Id and Name in the List of Merchants list.
4 Merchant Id Merchant identifier.
5 Name Name of the merchant.
6 Actions In this column you can either Edit or Delete a merchant. See Managing Merchants for more details.
7 Showing [number 1] to [number 2] of [number 3] entries Shows how many entries ([number 1] to [number 2]) are displayed in the table. [number 2] depends on the value set in Show [number] entries (2), [number 3] represents total number of entries in the merchants table.
8

Previous [page number] Next

[page number] is the number of page you are currently on. Here you can switch to the previous page or to the next page.

Managing Merchants

To create a merchant, do the flowing:

  1. Click +Add Merchant in the top right corner.

  2. Enter Merchant name and hit Save

    .

    3. The created merchant will appear on the Overview of Merchants page.

 

To edit a merchant, do the following:

  1. In the List of Merchants table (Overview of Merchants page), find Actions column and click Edit for a merchant you want to edit.
  2. On the Edit Merchant page, edit the merchant name and hit Save.

 

To delete a merchant, in the List of Merchants table (Overview of Merchants page), find Actions column and click Delete for a merchant you want to delete. The merchant will be deleted.
