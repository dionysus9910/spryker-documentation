Tutorial - CMS

used to be: http://spryker.github.io/challenge/cms/

Challenge Description

Create a static Contact Us page and integrate it into Yves. Then create your own template and use it.

Challenge Solving Highlights

Static page

For creating a static page, follow the steps:

  1. Go to Zet UI and open the CMS Pages backend. Add a CMS page that uses the URL /de/contact.
  2. Then add the text you would like to show on the static page.
  3. Activate the page.

The page is there! You can see the page here.

Templates

  1. Now to use your own templates, create a template under src/Pyz/Yves/Cms/Theme/default/template and call it contact.
  2. Add a placeholder and call it MyPlaceholder. You can open some other templates to see how it’s done.
  3. Go back to the CMS Pages backend and edit your page, then change the template to use contact.
  4. Add a new text to the placeholder MyPlaceholder.

Go back to the page and have a look on the changes.
