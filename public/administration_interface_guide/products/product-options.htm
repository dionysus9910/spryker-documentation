Product Options

To open this page, go to Products -> Product Options.

This page is where a shop administrator can add and manage product options, such as warranty and insurance, create product option groups and assign products to product options.

From the Product Options page, you can:

  • Create a new product option
  • View product options and their details
  • Edit product options
  • Activate or deactivate product options
Graphic User Interface

The Product option list page consists of the following elements:

The following table describes each element by their number:

# Element Description
1 + Create product option Creates a new product option. See Creating a New Product Option for more details.
2 Show [number] entries The element allows you to set the number of items you want to see in the list of product options.
3 Search: It allows searching product options details, such as Group name, SKU, and Name. Start typing and a list of matching items will be displayed in the table of Product option list.
4 Option group ID Identifier of the product option group.
5 Group name Glossary key of the product option group.
6 SKU SKU of the product option.
7 Name Glossary key of the product option.
8 Gross price Gross price of the product option.
9 Net price Net price of the product option.
10 Status Defines whether the product option group is enabled (active) or disabled (inactive).
11 Actions Using these buttons, you can Edit, View, or Activate/Deactivate a product option group. See Managing Product Option Details for more details.
12 Showing [number 1] to [number 2] of [number 3] entries The field shows how many entries ([number 1] to [number 2]) are displayed in the table. [number 2] depends on the value set in Show [number] entries (2), [number 3] represents a total number of entries in the table.
13 Previous [page number] Next It shows the number of the page you are currently on. Here you can switch to the previous page or to the next page.
Creating a New Product Option

To create a new product option:

  1. On the left-side navigation bar, select Products -> Product Options.
  2. Click +Create product option in the top right corner of the page.

    This opens the Create product option page containing two tabs: General Information and Products.
  3. In the General Information tab, enter and select the following details:
    • Group name translation key: Add a glossary key for a product option group.

      The format of the group name translation key should be as follows:
      product.option.group.name.[your key]. For example, product.option.group.name.test.

    • Tax Set: Define a tax set assigned to your product option group.
    • Option name translation key: Enter a glossary key for a product option value.

      The format of the option name translation key should be as follows:
      product.option.[your key]. For example, product.option.newtest.


      You can remove an option value by clicking Remove next to the Option name translation key and SKU fields.

    • SKU: Add a unique SKU for a product option value or proceed with the auto-generated one.
    • Prices: Specify gross and net prices for a product option value.

      If you want to add several product options values, click Add option below the Price section.


    • Translation: Add a group name and option name that will displayed in the shop application per each locale.

      You can copy the Group and Option names from one locale to another using the corresponding Copy icon.


  4. In the Products tab, select product(s) to be assigned to the product.

    Alternatively, you can click Select all on the page. In this case, all the products displayed on the page will be selected and added to the product option.
    However, if you do not want to assign the products, clear check-boxes next to the products you selected or click Deselect all on the page (this will remove all products from the to-be-assigned list you selected on this page).



    The products you select will appear in the Products to be assigned tab.
  5. Click Save.
Managing Product Options in Admin UI

You can edit, view, activate or deactivate the product option using buttons in the Actions column.

Editing Product Options Details

To edit a product option, click Edit in the Actions column on the Product option list page.

On the Edit product option page, you can modify the following information related to the option group or option values, such as price, tax set, group name and option name. Also, it is possible to remove the option or add new ones.

Keep in mind that Group name translation key, Option name translation key and SKU values cannot be updated.

Specifically, from this page, you can:

  • Add or remove option values by clicking Remove (1) or Add option (2) respectively.
  • Deactivate or activate product option group by clicking Deactivate or Activate in the top right corner of the page.

In the Products tab, you can select what products should be assigned to the product option or removed from it:

  • All products: Displays all available products. To assign a product to the product option, simply select the respective product(s). After that, the product(s) will appear in the Products to be assigned tab.
  • Assigned products: Displays what products have already been assigned to the product option.
    If you clear a check-box next to the assigned product, it appears in the Products to be deassigned tab.

    To remove products from the list of assigned ones, clear the check box or click Deselect all on the page (in this case, all check boxes will be cleared).

  • Products to be assigned: Shows products you have selected to be assigned to the product option.

    If you want to remove products from the Products to be assigned list, click Remove.

  • Products to be deassigned: Displays products you are going to remove from the assigned list of products.
    If you decide not to deassign products, click Remove in the Selected column.
Viewing Product Options

Clicking View will open the View product option page where you can view information related to product options: general information, gross and net prices, name displayed in the shop application, and products assigned to this option.

This page allows you to perform the following actions:

  • Go to the page where you can edit product option details. To do this, click Edit.
  • Deactivate or activate the product option group. For this, click Deactivate or Activate in the top right corner of the page.
  • Return to the list of product options by clicking List of product options in the top right corner.
Activating or Deactivating Product Option Group

You can enable (make visible in the shop application) or disable (make invisible in the shop application) a product option group.

To activate a product option group, click Activate in the Actions column of the Product option list.

To deactivate a product option group, click Deactivate. This will change the status to Inactive and make it invisible in the shop application.
