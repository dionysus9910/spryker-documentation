Attributes

To open this page, go to Products -> Attributes.

This page is where you can create and manage product attributes in your online shop.

Specifically, from this page you can:

  • Create product attributes
  • Manage product attributes
  • Add or remove predefined values of attributes
Graphic User Interface

To open Attributes, on the left-side navigation bar, click Attributes under Products.

The Product Attributes page displays the following elements:

In the table below, you can find descriptions of the screen elements by their number:

# Element Description
1 Create Product Attribute Creates a new attribute of the product. See Creating a New Product Attribute for more details.
2 Show [number] entries Here you can select how many items you want to see in the list of attributes.
3 Search This field allows searching product attributes. To find a specific attribute, start typing its name. As you type, a list of matching attributes will be displayed in the table.
4 Attribute ID Sequence number.
5 Attribute Key Unique identifier of an attribute.
6 Super Attribute Defines whether the attribute is a super attribute or not:
  • false means it is not a super attribute.
  • true means it is a super attribute.
7 Type Input type.
8 Actions This column allows you to click a respective button either to View or Edit attribute details. See the Managing Product Attributes section for more details.
9 Showing [number 1] to [number 2] of [number 3] entries The field shows how many entries ([number 1] to [number 2]) are displayed in the table. [number 2] depends on the value set in Show [number] entries (2), [number 3] represents a total number of entries in the table.
10 Previous [page number] Next It shows the number of the page you are currently on. Here you can switch to the previous page or to the next page.
Creating a New Product Attribute

To create a new product attribute:

  1. On the left-side navigation bar, go to Products -> Attributes.
  2. In the top right corner, click + Create Product Attribute.

    This will take you to the Products Attribute editor.
  3. Enter the required information:
    • Attribute key: Add a new attribute to the product.

      Keep in mind that it should not contain any special symbols.

    • Input type: Select the type for input data from the drop-down.
    • Super attribute: Select if you want to mark the attribute a super attribute.

      When this option is selected, the Allow input any value other than predefined option is disabled as a super attribute always uses predefined values.


    • Predefined Values: Add a new value (values) and click Enter.
    • Allow input any value other than predefined ones: Select if you want to add values different to the predefined ones when adding an attribute to the product or editing an existing attribute assigned to the product.
  4. Click Save.
  5. In the Translations tab, add the translation for your attribute key.

    If you want to apply this translation to other languages, click Copy to other languages next to the translation field.

  6. Select Translate predefined values if you want to add translations to predefined values. This step is optional.
  7. Click Save.
Managing Product Attributes

You can view or edit product attributes.

Viewing Product Attributes

To view a product attribute, click View in the Actions column. The View Product Attribute window opens. There you can view some general information on the attribute and its values, as well as check translations it has.

This page allows you to:

  • Return to the list of attributes by clicking Back to List in the top right corner of the page.
  • Go to the page where you can edit product attributes. To do this, click Edit in the top right corner of the page.

Editing Product Attributes

Here you can add or remove predefined values, and edit translations of the product attribute.

To edit a product attribute, click Edit in the Actions column. This will take you to the Edit Product Attribute page.

Attribute key, input type, and super attribute are disabled, as they cannot be edited.

From this page you can return to the list of product attributes by clicking Back to List in the top right corner of the page.
