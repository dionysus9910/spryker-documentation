Content Widgets

CMS widgets allow creating blocks, charts, a dynamic listing of product items, a product set or group, as well as embedding media files into CMS pages or blocks. This article describes how to add and use content widgets in your shop.

CMS Widget Overview

In Zed, a shop administrator can add the following CMS content widgets:

  • cms_file
  • product_set
  • product_group
  • cms_block
  • chart

To work with CMS content widgets, the Content Widget feature must be enabled for your project.

To access a CMS widget:

  1. On the left-side navigation bar, navigate to the Content Management menu and select Pages.
  2. Click Edit → Placeholders next to the page on which you want to position a widget. This will take you to the Edit Placeholders: page name editor where you can add widgets to your website.
  3. In the Content tab, place a mouse pointer on the defined content field.
  4. Click content widget and select a widget from the drop-down list. This will embed the following template into the content field:

    5. {{ function name (['identifier']) }}.
    You need to replace a function name with the respective widget name and identifier - with parameters of the function.

Below we will consider each CMS widget: how you, as a shop administrator, can manage it in Zed and how your shop visitors can view it on Yves.

Adding CMS widgets to a CMS page in the Admin UI

CMS widget: cms_file

Before proceeding, make sure that the File Uploader feature is integrated into your project.

This widget enables you to attach media files to a CMS page. Each file is displayed as a download link. Clicking on it will download files such as videos, guides, images, etc from the website to the device.

Keep in mind that files should be first uploaded to File list under File Manager in Zed. See Media Management / File Uploader to learn how to to do this.

To add a cms_file content widget:

  1. Select cms_file from the content widget menu.
    This will place the following string {{ cms_file(['identifier']) }} where identifier is the file ID you need to insert.

    If you want to add several files to your page, insert them wrapped into brackets and single quotes and separated by a comma. For example: {{ cms_file(['identifier'], '[identifier'], '[identifier']) }}

    You can look for the file ID in File List. For this, go to File Manager → File List. On the Files list page that opens, the # column is the file ID.
  2. To save the changes, click Save at the bottom of the page.
  3. Click Preview to view the way your content would render on the page on Yves.

To make your changes visible on Yves, click Publish in the upper right corner of theEdit Placeholders: page name page.

CMS widget: product_set

Before proceeding, make sure that the Product Sets feature is integrated into your project.

The widget can place a collection of products based on their specific characteristic. Clicking Add all to cart will allow end users to purchase all the product items listed in the set.

To enable Add all to cart and Add to cart, make sure that the product is available on the stock.

To place a product set on a CMS page:

  1. Select product_set on the content widget menu.
    This will add the following code: {{ product_set(['identifier']) }} where identifier is a key of a product set.

    The product set key must be active. Otherwise, it will not be displayed on Yves.

  2. Insert this key instead of an identifier.
    To access the key:
    1. On the left-side navigation bar, click Products and select Product Sets. The Product Sets page opens.
    2. Click Edit next to the product set you want to add.
    3. In the General tab → Product Set Key field, copy the set_key value.

  3. To save the changes, click Save at the bottom of the page.
  4. Click Preview to view the way your content would render on Yves.

To make your changes visible on Yves, click Publish in the upper right corner of the Edit Placeholders: page name.

The page with the widget is displayed on the website.

CMS widget: product_group

Before proceeding, keep in mind that the Product Group feature is integrated into your project.

You can use this widget to group different products by a specific attribute, for example, color, memory, or size. The products will be displayed as a grid on the page. Clicking the product will take your customers to the respective product detail page.

To add a widget:

  1. Select product_group on content widget.
    This will place the following string to the content area: {{ product_group(['identifier']) }}.
  2. Replace the identifier with a product (abstract) SKU.
    You can get it from the Product list. To access it, on the left-side navigation bar, click Products -> Products.
  3. To save the changes, click Save at the bottom of the page.
  4. Click Preview to view the way your content would render on Yves.

To make your changes visible on Yves, click Publish in the upper right corner of the Edit Placeholders: page name page.

Below you can see the way your widget will be placed on the page on Yves.

CMS widget: cms_block

Before proceeding, keep in mind that the Content Widget CMS block feature is enabled for your project.

This widget embeds multiple blocks containing some content into the same page. This can be required when, for instance, you need to hold some season or promotional campaigns or want to create banners on a page.

To add a CMS block widget to the page, follow the instructions described in the step Adding a CMS Block to a CMS Page [add a link].

CMS widget: chart

Before proceeding, make sure that the Chart feature is enabled for your project.

Charts allow attracting your visitors by visualizing data and making the content understandable. The widget easily adds charts and graphs to your CMS page.

To place a chart content widget on a CMS page:

  1. Select chart from the content widget menu.

    2. This will add the following string {{ chart('identifier') }} where identifier is the name of the plugin containing output data.
  2. Insert the respective identifier.
  3. To save the changes, click Save at the bottom of the page.
  4. Click Preview to view the way your content would render on Yves.
To make your changes visible on Yves, click Publish in the upper right corner of the Edit Placeholders: page name page.

Below you can see the way your widget will be placed on Yves.

Viewing CMS Widgets on Yves

You can also view the page how it renders on Yves from the Zed dashboard. For this, go to Content Management → Pages and select View → In Shop next to the respective page.
