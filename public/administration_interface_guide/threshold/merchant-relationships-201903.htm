Merchant Relationships

To open this page, go to Thresholds -> Merchant Relationships.

This page allows you to set up and check thresholds for minimum order value within a specific merchant relationship.

From this page you can:

  • Edit the merchant relationship
  • Configure the hard threshold for the merchant relationship
  • Configure the soft threshold for the merchant relationship

Graphic User Interface

Hover your mouse over the numbers to view their description.

Managing Merchant Relationship Thresholds

Editing Merchant Relationship Threshold

Every merchant relationship can have both hard and soft threshold. The thresholds need to be configured for every store and currency.

To edit the merchant relationship threshold, click Edit in the Actions table.

Configuring the Hard Threshold for the Merchant Relationship

With a hard threshold applied, a customer cannot proceed to checkout if the buyer's cart value is below the minimum order value specified below.

To configure the hard threshold, do the following:

  1. In the Store and Currency drop-down list box, select the type of the store and currency you are going to create the threshold for.
  2. Enter the minimum order value for the hard threshold.

    The shopper will not be able to proceed to checkout if their cart value is below the minimum order value set in this box.

  3. Enter the text for messages for each locale defined in the shop. The messages will appear when the cart value is not reached.

    Example
    Example of the message: You should add items for {{threshold}} to pass a recommended threshold. You can't proceed with the checkout.

Configuring the Soft Threshold for the Merchant Relationship

A soft threshold allows a customer to proceed to checkout, but displays a flash warning message and adds an additional fee to the order sum as a penalty that the customer will need to pay.

To configure the soft threshold, do the following:

  1. In the Store and Currency drop-down list box, select the type of the store and currency you are going to create the threshold for.
  2. Select the appropriate option in the Soft Threshold settings.

    Depending on the selected threshold configuration, follow the steps described in the Type Specific Steps column and then proceed to step 3.

    Soft Threshold TypeType Specific Steps
    NoneN/A
    Soft Threshold with messageEnter the value into the Minimum Order Value field.
    Soft Threshold with fixed fee
    1. Enter the value into the Minimum Order Value field.
    2. Enter the fixed fee value into the Fixed Fee field. This amount will automatically be added once the cart total doesn't reach the amount specified in the Minimum Order Value field.
    Soft Threshold with flexible fee
    1. Enter the value into the Minimum Order Value field.
    2. Enter the flexible fee value into the Flexible Fee field. The value is the percent that is calculated from the cart grand total in case the threshold specified in the Minimum Order Value is not reached.
  3. In the Message field, enter the message text for the corresponding store locales.
  4. Click Save.
