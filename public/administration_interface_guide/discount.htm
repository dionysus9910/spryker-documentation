Discount

To access this page, go to Discount.

On this page, you can define discounts for products. Discounts can either be cart-based, or they can be in form of vouchers.Cart-based discounts allow you to create and apply special rules to cart content. For example, automatically applying a discount for purchases over a certain sum. Vouchers are codes that can be given to customers to be redeemed during checkout. Voucher codes are grouped into pools to apply logic to multiple vouchers at once. You can generate a single voucher that can be can be used by multiple customers or a pool of dedicated one-time voucher codes per customer.

Discounts in general can be set to be "exclusive". An exclusive discount used on its own. All other discounts cannot be applied with an exclusive discount unless a higher exclusive discount provided. Then, the higher discount is redeemed. Non-exclusive vouchers can be redeemed in conjunction with other non-exclusive discounts.

In terms of the way a discount is calculated, there are two types of discounts: percentage discount and fixed amount discount.

From this page you can:

Create a discount voucher

Create a cart rule discount

Creating a Discount Voucher

To create a voucher you will need to define the general information,(e.g. name, validity period), the discount rules and items and when you want the discount to be applied.

The following steps will help you create a voucher for your shop:

1.

On the left navigation bar, select Discount
 

2.

Click Create new Discount from the top right corner
3.

Select discount type and enter name and description

 

In discount type select Voucher codes. The name and the description should be meaningful to help others using the back-office application understand what the discount does.

Discount Name
The given name is displayed in the customer's cart when redeeming the voucher and therefore, it must be unique.
4. Set if the voucher is exclusive or not.
5. Setup the validity interval (lifetime) of the voucher.

 

Vouchers are redeemable between Valid From and Valid Until dates, inclusive.

In the example below, the voucher can be redeemed starting from 1/1/2017 until 31/12/2017.
6. Switch to the tab Discount Calculation The discount can be calculated in two ways:

 
  • Percentage Value: the discount is calculated as a percentage from the discounted items
  • Fixed Value: a fixed amount is discounted

7.

Define the conditions on which the discount can be applied

 

You can use a query to define discounts conditions. Only products that satisfy the query's conditions are discountable. Queries also define if the discount is applied to one or many products.

Use the Query Builder, to construct queries (guided) or the simple input field (free text).You can freely switch between both modes as you like (Note: incomplete queries cannot be transferred between the two modes).

Query Builder & Plain Query
The query builder lets you combine different conditions with connectors (AND and OR). Multiple conditions (rules) can be added and grouped in this way. Each condition (rule) consists of a field (e.g. attribute.color), an operator (e.g. equal: =), and a value (e.g. blue). The fields and values are defined by your shop data.

  Description

Fields

The available fields may include SKU, item-price, item-quantity or a variety of attributes (e.g. attribute.color).

Operator

The operator compares the value of a field on the left with the value(s) on the right (e.g. equals ‘=’, greater than ‘>’). If the expression evaluates to true, the discount can be applied.

Value

The value types must match the selected field. The asterisk (*) matches all possible values.

Combine Conditions

‘AND’ and ‘OR’ operators are used to combine conditions.

Grouping

When building more complex queries, conditions may grouped inside brackets ‘( )’

8. Add a set of conditions for your new voucher

 

Vouchers can be linked to one or more conditions (also called decision rules). Vouchers are only redeemed if all linked conditions are satisfied.

Again, the conditions are created in the form of a query and may be entered as a plain query or via the query builder.

If you don’t need to add a condition, leave the query builder empty.
9. Click Save to generate the new voucher codes. They will appear in the voucher codes tab.

 

Here you can generate and view voucher (if they were already created). The list is empty until codes are generated.
10. Enter the number of voucher codes you want to generate

 

You can generate a single voucher code or multiple voucher codes.
11. Enter a Custom Code

 

Use the placeholder [code] to indicate the position you want the random characters to be added.

If you want to create a single voucher code,you can manually create the code. If you want to create multiple voucher codes at once, you need to add a "Random Generated Code Length" to the custom code (next step).
12. Set the Random Generated Code Length

 

Only relevant for generating multiple distinct voucher codes, this will add a random number of characters to the entered custom code.

Set the value to the codes you want to generate.
13. Set the Maximum number of uses

 

Define the maximum number of times a voucher code can be redeemed in a cart.
Value Behavior

0

Infinitely redeemable

1

The voucher can be redeemed once

n > 1

The voucher can be redeemed n times.

Voucher Code Pool
The voucher codes of a discount are all contained in the same voucher code pool. One customer may only redeem one voucher code per pool per cart.

14.

Click, Generate to complete this process.

 

The voucher codes will be generated according to your specifications. and a list of codes will be displayed in a table at the bottom of the page in the table.
15. Click, Save and activate your voucher.

 

Even if a voucher is valid and the decision rules are satisfied, a voucher can only be redeemed if it’s currently active.

Creating a Cart Rule Discount

A cart rule is a discount that is contained in the cart and needs no input from the customer. If the discount is active, valid and the cart satisfies the linked conditions, then it’s automatically calculated for the order.

Creating a Cart Rule
The steps for creating a cart rule are similar to the ones for creating a voucher, except you don't need to generate voucher codes for it.

You can follow the steps described below to create a new cart rule discount.

1.

On the left side navigation bar, select Discount

2.

Click on the Create new Discount button in the top right corner

3.

Select the discount type and enter a name and description.

As a discount type select Cart Rule. The name and the description should be meaningful for the ones using the back-office application.

Discount Name
If all the conditions are fulfilled and the cart rule is applied, the given name will be displayed to the customer.

4.

Set up if the cart rule is exclusive

The exclusivity option conditions if the discount can be combined or not.

Exclusive Discounts
An exclusive discount will be applied only alone. It discards all other discounts unless there is a higher discount provided by another exclusive discount. In that case, the exclusive discount that offers a higher discounted value is redeemed. Non-exclusive discounts can be redeemed in combination with other non-exclusive discounts.

5.

Setup the validity interval for the cart rule

The cart rule will be active starting with the Valid From date until the Valid Until date inclusive.

For the example below, the cart rule is active starting from 1/1/2017 until 31/12/2017.

6.

Switch to the tab Discount Calculation

The discount can be calculated two ways:

  • Percentage Value: the discount is calculated as a percentage from the discounted items
  • Fixed Value: a fixed amount is discounted

A cart rule can have one or more conditions (also called decision rules) linked to it. The cart rule can be redeemed if and only if every condition linked to it is satisfied. Again, the conditions are created in the form of a query and may be entered as a plain query or via the query builder.

If you don’t need to add a condition, you can leave the query builder empty.

Example: Discount is applied if five or more items are in the cart and it is Tuesday or Wednesday.
